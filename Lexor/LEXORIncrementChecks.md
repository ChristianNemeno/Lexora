LEXOR Increment Checks

[Increment 1]
- Should be able to recognize basic structure of LEXOR interprter
- Can recognize comment
- Able to recognize reserved words
- Able to concatenate
- Should be able to declare variables
- Should be able to assign data to a variable
- Able to display (PRINT)

[Increment 2]
- Can recognize unary operator
- Should be able to read input data (SCAN)
- Should be able to perform arithmetic and logical operations

[Increment 3]
- Can execute IF and IF ELSE control structure (including nested)

[Increment 4]
- Can execute FOR (including nested)
- Can execute REPEAT (including nested)

[Increment 5]
- Can do while loop
